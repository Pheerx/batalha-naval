Jogo Batalha Naval em C

Este projeto implementa um jogo de Batalha Naval simplificado em linguagem C, onde:

  • Um tabuleiro 10x10 é representado por uma matriz bidimensional.
  • São posicionados quatro navios no tabuleiro (2 horizontais/verticais e 2 diagonais).
  • São implementadas três habilidades especiais com áreas de efeito distintas: Cone, Cruz e Octaedro.
  • As áreas de efeito das habilidades são sobrepostas no tabuleiro e exibidas de forma clara no console.

Como usar o código

Compile o programa usando o GCC:

gcc batalha_naval_com_habilidades.c -o batalha_naval

E execute:

./batalha_naval

Legenda da saída do programa

  • ~: Água
  • #: Navio
  • *: Área afetada pela habilidade
